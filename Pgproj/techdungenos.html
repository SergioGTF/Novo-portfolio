Sobre o Projeto

Tech Dungeons é um jogo de aventura/puzzle onde o objetivo é o jogador conseguir passar de três fases. Em cada fase, o jogador deve se movimentar para pegar uma chave e abrir a porta fechada.

O jogo foi desenvolvido em C e utiliza conceitos de lógica de programação e estrutura de dados.
